## 📦 Installation
```bash
# Clone
git clone https://github.com/raytp29-hub/big-ambitions-analyzer1.0.git
cd big-ambition-analyzer

# Virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install
pip install -r requirements.txt

# Run
streamlit run app.py
